EP1 -- Perceptron: toy example¶

Seja um conjunto de treinamento $X = \{ (\mathbf{x}^{(1)},y^{(1)}), (\mathbf{x}^{(2)},y^{(2)}), \ldots, (\mathbf{x}^{(N)},y^{(N)})\}$, $\mathbf{x}^{(i)} \in \mathbb{R}^d$ e $y^{(i)} \in \{-1, +1\}$, $i=1,2,\ldots,N$, linearmente separveis. Dado $\mathbf{x} = (x_1, \ldots, x_d) \in \mathbb{R}^d$, seja $\tilde{\mathbf{x}} = (1, x_1, \ldots, x_d) \in \mathbb{R}^{1+d}$.

O perceptron determina um vetor de pesos $\mathbf{w} \in \mathbb{R}^{1+d}$ de tal forma que $\mathrm{sign}(\mathbf{w}^T \tilde{\mathbf{x}}^{(i)}) = y^{(i)}$, $\forall i$, na qual $$ sign(z) = \left\{\begin{array}{ll}+1, & \mbox{se $z \geq 0$,}\\ -1, & \mbox{se $z < 0$.}\\ \end{array}\right. $$

O objetivo deste EP implementar o algoritmo perceptron para dados em $\mathbb{R}^2$ e test-lo em casos com poucos exemplos.

1. Gerar os pontos e uma target function¶

  • o dataset consistir de alguns ($N$) pontos definidos a mo. Alungs desses pontos sero os vrtices do quadrado $[0,1]\times[0,1]$
  • definiremos uma reta $f(x_1,x_2) = w_0 + w_1\,x_1 + w_2\,x_2$ que corta o quadrado $[0,1]\times [0,1]$
  • aqui vamos plotar esses pontos de tal forma que os pontos $(x_1,x_2)$ tais que $f(x_1,x_2) \geq 0$ (positivo) so plotados em azul e os pontos $(x_1,x_2)$ tais que $f(x_1,x_2) < 0$ (negativo) so plotados em vermelho
  • a classe de cada um dos $N$ pontos ser portanto dado pelo sinal de $f$

    # misclassifications

def perceptron(Xe,y,w0):
    """
    Parameters:
       Xe : ndarray (N,d+1) - it already has the 1's in column 1
       y  : ndarray (N,)
       w0 : ndarray (d+1,) - the initial weight vector
       
    Returns:
       w : ndarray (d+1,) - the final weight vector
    """
    d = len(y)
    cont = 0
    ind = 0
    # write your PERCEPTRON algorithm code here
    while (cont < d):
        signal = np.sign(np.dot(Xe,w0))
        if signal[ind] == y[ind]:
            cont+=1
            ind = (ind + 1)%d
        else:
            w0 = w0 + y[ind] * Xe[ind]
            cont = 0
            ind = (ind + 1)%d
    return w0
